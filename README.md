# quizcode-back

Backend de QuizCode, una aplicación para la creación y gestión de cuestionarios de programación con evaluación asistida por inteligencia artificial.

Desarrollado con Spring Boot y Java. Utiliza MongoDB como base de datos, Spring Security con JWT para la autenticación, y Spring AI (compatible con Groq y Gemini) para la generación y corrección automática de preguntas mediante IA.
